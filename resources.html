How to use HS well

Log same day

Daily entry is the whole game. Add every spend and income the same day so totals stay trustworthy.

Use categories consistently

Examples: Food, Salary, Travel, Rent, Recharge, Medicine. Consistent categories make the ledger easier to scan.

Track udhar separately

Use Loan Given and Loan Returned for money that should come back. This keeps income and loans from getting mixed up.

Always keep a backup

Use Gmail backup or JSON export regularly, especially before changing phones or resetting storage.
