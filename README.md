# Hisaab Sathi

Hisaab Sathi is the finance ledger app for the Aapka-Sathi family.

## Features

- daily expense logging
- income logging
- udhar given and returned tracking
- mobile-friendly ledger and summary view
- shared Google login with family Firebase
- cloud sync and Gmail backup fallback
- PWA install support
- day/night theme
- reminder and feedback flows
